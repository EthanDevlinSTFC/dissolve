Structure Refinement for Scattering Data

{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Dissolve is a simulation tool for the interrogation of neutron scattering data, typically total neutron scattering data. It builds on the techniques established in the Empirical Potential Structure Refinement (EPSR) method by Soper. Dissolve is in heavy ongoing development, and as such the code here is rather volatile.
